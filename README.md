\# OpenGL SmartWatch



Projekat pametnog sata napravljen u OpenGL-u za kurs Računarska grafika, FTN.



\## Funkcionalnosti

\- Prikaz vremena

\- EKG animacija sa animiranim srcem

\- Indikator baterije

\- Navigacija između ekrana (strelice levo/desno)

\- Digitalni prikaz cifara



\## Tehnologije

\- C++

\- OpenGL

\- GLEW 2.2.0

\- GLFW 3.4.0



\## Pokretanje

1\. Otvori `SmartWatch.sln` u Visual Studio

2\. Build (Ctrl+Shift+B)

3\. Run (F5)
